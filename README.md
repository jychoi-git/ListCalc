#  최제영의  ListCalc 저장소

## 프로젝트 소개
자연수들의 리스트연산을 할 수 있습니다.

## 개발기간
2025.9.1. ~ 2025.12.31

## 개발자 구성
* 팀장 : 홍길동(jychoi-git) : front-end 개발
* 팀원 : 임꺽정() : back-end 개발
* 
## 개발 환경
Python 3.8

## 주요 기능
리스트연산(0/1/2/3/4)을 선택하고 자연수 2 개 이상을 입력하면 해당 연산 결과를 출력합니다.

## 프로그램 사용법
`python calc.py`
